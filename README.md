# Playwright Automation Framework

## Setup
1. Clone repo ini.
2. Jalankan `npm install`.
3. Copy `.env.example` menjadi `.env` dan isi kredensial kamu.
4. Jalankan `npx playwright install`.

## Cara Eksekusi
- **Jalankan semua test**: `npm test`
- **Lihat report Allure**: `npm run allure`
- **Jalankan satu file saja**: `npx playwright test tests/login.spec.ts`

## Instalasi
1. Install Node.js: Pastikan laptop kamu sudah terinstal Node.js (minimal versi 18)
